# hexo d 后 Custom domain 被置空

Hexo 博客部署在 GitHub 上，当手动填写了 Custom domain 后，在下一次 hexo d 后会被置空。
解决方案是：在 Hexo 目录下的 source 文件夹下创建名称为 CNAME 得文件，将域名 blog.qmzbe.top 写在里面（只能写一个）。然后执行

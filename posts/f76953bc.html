随机壁纸

网址：https://api.btstu.cn/doc/sjbz.php

请求方式： GET

请求地址： https://api.btstu.cn/sjbz/api.php

返回格式： json/images

请求示例： https://api.btstu.cn/sjbz/api.php?lx=dongman&format=images

请求参数：

名称 必填 类型 说明
method 否 string 输出壁纸端[mobile
lx 否 string 选择输出分类[meizi
format 否 string 输出壁纸格式[json

返回参数：

名称 类型 说明
code string 返回的状态码
imgurl string 返回图片地址
width string 返回图片宽度
height string 返回图片高度
